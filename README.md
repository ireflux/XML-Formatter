# XML 格式化工具 (XML-Formatter)

一个简单易用的在线 XML 格式化工具，支持 XML 代码的格式化和压缩功能。

## ✨ 功能特点

- 📝 XML 格式化：自动缩进和美化 XML 代码
- 🗜️ XML 压缩：移除多余空白字符，生成紧凑的 XML
- 📋 一键复制：快速复制处理后的代码
- 💻 响应式设计：完美支持桌面和移动设备
- 🚀 实时处理：即时显示处理结果
- 🛡️ 错误提示：友好的错误提示和验证

## 🚀 使用方法

1. 在文本框中输入或粘贴需要处理的 XML 代码
2. 点击"格式化"按钮美化代码，或点击"压缩"按钮压缩代码
3. 使用"复制"按钮复制处理后的代码

## 🛠️ 技术实现

- 纯前端实现，无需后端服务
- 使用原生 JavaScript 实现 XML 解析和处理
- 采用现代 CSS 实现响应式布局
- 支持大文件处理（超过 10 万字符会提示性能警告）

## 📝 注意事项

- 确保输入的 XML 代码格式正确
- 处理大文件时可能需要等待较长时间
- 建议使用现代浏览器访问以获得最佳体验
