## Product Overview

XML 格式化工具 (XML-Formatter) is a simple, browser-based XML formatting and compression tool.

### Core Features

- XML formatting with automatic indentation (4 spaces)
- XML compression to remove whitespace
- One-click copy functionality
- Real-time processing with error validation
- Responsive design for desktop and mobile
- Performance warnings for large files (>100k characters)

### User Experience

- Pure frontend implementation, no backend required
- Instant feedback with toast notifications
- Supports XML declarations, DOCTYPE, and comments
- Chinese language interface (zh locale)
